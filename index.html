StreetGuessr setup

This game uses the Google Maps JavaScript API for the 360° panorama and map. Paste your API key below, then click Load Game.

  1. Enable Maps JavaScript API in a Google Cloud project.
  2. Enable billing for that project.
  3. Use a browser-restricted API key for real deployment.

Tip: for local testing, run a tiny local server from this folder: python -m http.server 8000, then open http://localhost:8000/index.html.

Ready to guess?

Click Start Game. You will get a random 360° Street View panorama. Rotate the view, click the map to guess, then submit before the timer ends.

How to play

Use the Street View window to inspect clues. Click the map to place your guess.

Your score is based on distance: exact or near-exact guesses can earn up to 5,000 points.
